# NomsFood
A Web Application where vendors are able to list their surplus food, and users will be able to purchase food at discounted prices

List of Features:
Vendor Web Portal
- Profile management
- List Food Listings
- Order tracking
  
User Web Portal
- Profile management
- Search for stores
- Place an order
- Check out & Payment(Paypal integration) 

Admin Web Portal
- Profile management
- Manage all users
- Resolve IT Tickets
